-------------------------------------------
SYSTEM OVERVIEW
-------------------------------------------
Aries is an advanced, human-like AI assistant designed to interact naturally,
recognize emotions, recall memories, and assist in real-time tasks.
It combines powerful LLMs, speech recognition, TTS, and memory systems.

Core Components:
- Large Language Model (Mistral 7B / Phi-3 Mini)
- Speech Recognition (OpenAI Whisper)
- Text-to-Speech (Kokoro TTS)
- Emotion Detection (MFCC + SVM/MLP)
- Contextual Memory System (Google Drive JSON storage)
- Spontaneous Conversations & Recall
- Web Interface (FastAPI/Flask + WebSocket)
- Real-Time Audio Input/Output

-------------------------------------------
INSTALLATION STEPS
-------------------------------------------
1. Clone the Repository:
   git clone https://github.com/Aoi-03/Aries.git
   cd Aries

2. Install Requirements:
   pip install -r requirements.txt

3. Configure Google Drive:
   - Add your service account JSON in /config
   - Enable Google Drive API

4. Run Aries:
   python app.py
